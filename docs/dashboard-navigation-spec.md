# ダッシュボード ナビゲーション改善・プレビュー機能追加 仕様書

## 1. 概要

マニュアル配信ダッシュボードのステップ間ナビゲーションの改善と、LINEメッセージプレビューの別ウィンドウ表示機能の追加。

## 2. 現状の課題

- Step2「記事選択」に明示的な「戻る」ボタンがない（「再取得」のみ）
- 各ステップのナビゲーションUI（戻る/次へ）が不統一
- プレビュー画面をダッシュボード内でしか確認できない
- プレビューを別画面で開いて詳細確認・共有する手段がない

## 3. 改善仕様

### 3.1 全ステップ共通ナビゲーション

各ステップの下部に統一的なナビゲーションバーを配置：

- 左側: 「戻る」ボタン（前のステップに遷移。Step1では非表示）
- 右側: 「次へ」ボタン（次のステップに遷移。最終ステップでは「配信実行」）
- ボタンスタイル:
  - 戻る: テキストリンク風（bg-white border text-slate-600）
  - 次へ: 緑ボタン（bg-green-600 text-white）

### 3.2 各ステップの戻るボタン動作

| ステップ | 戻り先 | 動作 |
|---------|--------|------|
| Step1 記事一覧取得 | なし | 戻るボタンなし |
| Step2 記事選択 | Step1 | 記事リストをクリア、最初からやり直し |
| Step3 テンプレート選択 | Step2 | テンプレート選択を保持したまま記事選択に戻る |
| Step4 確認・配信 | Step3 | 配信確認画面からテンプレート選択に戻る |

### 3.3 プレビュー別ウィンドウ機能

Step4の確認画面に「別ウィンドウでプレビュー」ボタンを追加：

- ボタンクリックで `/dashboard/preview` を新しいウィンドウで開く
- プレビューページはURLクエリパラメータでデータを受け取る（長文はlocalStorage経由）
- プレビューページの内容:
  - 白背景のフルスクリーン
  - 5種類全テンプレートを一覧で並べて表示（比較可能）
  - 各テンプレートのスマホフレーム付きプレビュー
  - 現在選択中のテンプレートにハイライト表示
  - テンプレート切替ボタン（クリックで選択変更）
  - 印刷/スクリーンショット対応のクリーンなレイアウト

### 3.4 プレビューページ仕様（/dashboard/preview）

- URL: `/dashboard/preview`
- データ受け渡し: `localStorage.setItem('linemag-preview', JSON.stringify(data))` でダッシュボードから渡す
- データ構造:
  ```json
  {
    "summaryTitle": "キャッチタイトル",
    "summaryText": "要約テキスト",
    "thumbnailUrl": "https://...",
    "articleUrl": "https://...",
    "articleCategory": "EC戦略",
    "selectedTemplateId": "daily-column"
  }
  ```
- 機能:
  - 全5テンプレートのプレビューを3カラムグリッドで表示
  - 選択中テンプレートに緑枠ハイライト
  - テンプレート名とカテゴリの表示
  - 「閉じる」ボタン
