# LineMag 統合仕様書 v1.0

> **作成日**: 2026-03-24
> **プロジェクト**: LineMag — LINE マガジン配信 + CRM + EC連携 + 予約 統合プラットフォーム
> **技術スタック**: Next.js 14 / TypeScript / Supabase / Vercel / LINE Messaging API / Gemini AI

---

## 概要

LineMagは、特定ブログサイト（meetsc.co.jp/blog/）の記事をAI要約してLINE公式アカウントから配信するシステムとして稼働中です。本仕様書は、既存機能を基盤として以下の機能群を統合的に拡張するための包括的な設計ドキュメントです。

### 拡張する機能群
1. **CRM（顧客関係管理）** — 顧客プロファイル・セグメント・ライフサイクル管理
2. **EC連携 + クーポン** — 自社ECサイト連携、クーポン作成・配布・利用管理
3. **予約システム** — 相談窓口のタイムスロット予約、Google Meet連携
4. **マーケティングオートメーション** — パーソナライズ配信、シナリオ配信、A/Bテスト
5. **分析ダッシュボード** — KPI可視化、レポーティング
6. **Supabaseへのデータベース移行** — ファイルベース保存からPostgreSQLへ
